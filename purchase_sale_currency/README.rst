======================
Purchase Sale Currency
======================

This module allows to:

* Create Request Of Quotations using the origin Sale Order currency.

Configuration
=============

Any special config is required.

Usage
=====
    * Create a Sale Order with stockable products.
    * Confirm the sale order and go to purchase module to see the corresponding purchase order with sale order currency.
